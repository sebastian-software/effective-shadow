Effective Shadow

Beautiful, multi-layered CSS shadows using Bézier curves.

Shadows are essential for creating depth and hierarchy in modern interfaces. But most CSS shadows look flat and artificial. Effective Shadow generates natural-looking, multi-layered shadows that mimic real-world light — giving your UI that polished, professional feel.

Elevation Levels

These are pre-generated, ready-to-use shadow levels — carefully tuned for visual harmony and optimized for performance. For most projects, these five levels provide all the depth you need: from subtle card shadows to dramatic modal overlays.

Need something custom? The JavaScript API lets you create your own shadows with full control over layers, blur, offset, and opacity curves.

Why Bézier Curves?

Most shadow generators use linear interpolation between layers — each step increases blur and offset by the same amount. This creates shadows that feel mechanical and artificial.

Effective Shadow uses Bézier curves to control how shadow properties change across layers. Just like easing functions make animations feel natural, Bézier-curved shadows create depth that mimics real-world light behavior:

  • Sharp near the surface — early layers have minimal blur
  • Soft further away — outer layers diffuse more naturally
  • Consistent opacity falloff — no harsh transitions between layers

The result? Shadows that feel like they belong in physical space, not painted on.

The Problem with box-shadow

CSS box-shadow only works on rectangular boxes. For icons, PNGs with transparency, or any non-rectangular element, it creates an ugly rectangular shadow around the bounding box:

Effective Shadow generates both box-shadow and filter: drop-shadow() variants. The filter version follows the actual shape of your element — perfect for icons, logos, and images.

We automatically compensate for the visual differences between these two CSS properties (drop-shadow uses Gaussian blur, box-shadow uses box blur), so your shadows look consistent regardless of which variant you use.

Quick Start

CSS Classes

The simplest approach — import our CSS and use class names directly:

import "@effective/shadow/shadows.css"
···
<div class="shadow-3">Elevated card</div>
<img class="drop-shadow-2" src="icon.svg" />

CSS Modules

For scoped styles with automatic class name hashing:

import shadows from "@effective/shadow/shadows.module.css"
···
<div className={shadows["shadow-3"]}>Elevated card</div>

Tailwind CSS Plugin

Extend Tailwind with our shadow utilities:

// tailwind.config.js
import effectiveShadow from "@effective/shadow/tailwind"

export default {
  plugins: [effectiveShadow]
}
···
<div class="shadow-effective-3">Card</div>
<img class="drop-shadow-effective-2" src="icon.svg" />

JavaScript API

For full control, use the TypeScript/JavaScript API:

// Pre-generated shadows (levels 0-5)
import { boxShadow, dropShadow } from "@effective/shadow"

element.style.boxShadow = boxShadow[3]
element.style.filter = dropShadow[2]
···
// Custom shadows with full control
import { buildShadow, toBoxShadow, toDropShadow } from "@effective/shadow"

const shadow = buildShadow({
  shadowLayers: 4,
  finalOffsetY: 12,
  finalBlur: 24,
  finalAlpha: 0.25
})

const css = toBoxShadow(shadow)
const filter = toDropShadow(shadow)
