Quick Setup

1. Install the package

npm install @effective/shadow

2. Add to tailwind.config.js

import effectiveShadow from "@effective/shadow/tailwind"

export default {
  plugins: [effectiveShadow]
}

3. Use the classes

<div class="shadow-effective-4">...</div>
